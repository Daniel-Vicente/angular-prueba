Desde el inicio
En CMD/DOS
npm install -g @angular/cli
en una carpeta de Escritorio ng new test-angular
si da error version npm 7 npm install --global npm@6
despues de instalado code . (o abrir desde visual la carpeta test-angular)
En terminal de Code levantar la app ng serve -o
en otra terminal (ya que esa esta corriendo el server) npm install bootstrap --save
npm install jquery --save
npm install bootstrap --save
npm install font-awesome --save (opcional, es para iconos)

Lineas modificadas en angular.json

        	"styles": [
          	"../node_modules/bootstrap/dist/css/bootstrap.min.css",
          	"../node_modules/font-awesome/css/font-awesome.css",  (opcional, es para iconos)
          	"src/styles.css"
        	],
        	"scripts": [
          	"../node_modules/jquery/dist/jquery.min.js",
          	"../node_modules/bootstrap/dist/js/bootstrap.min.js"
        	]
